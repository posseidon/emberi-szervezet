## File Organization

Each module is organized in its own directory with the following structure:

```
/history/{topic_id}/{module_id}/
├── {module_id}.md          # Main module content
├── flashcards/             # Study materials
└── quizzes/                # Assessment materials
```

## File Access

To access a specific module:
```
/history/{topic_id}/{module_id}/{module_id}.md
```

### Example Paths
- `/history/okor/athen-demokracia/athen-demokracia.md` - Ancient Greek Democracy
- `/history/kozepkor/aranybulla/aranybulla.md` - The Golden Bull
